title: 'Create an account'
openapi: 'POST /accounts'
---

<Info>
This endpoint requires a valid authorisation for the user. If no active authorisation exists, the request will be rejected.
You can optionally pass the `x-authorisation-id` header to specify which authorisation to use; otherwise the system will resolve a valid authorisation automatically.
See [Authorisations](/authorisations) for more details.
</Info>
